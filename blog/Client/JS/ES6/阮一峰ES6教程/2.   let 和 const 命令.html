总结：

1.var 和function()都存在变量提升，并且，var只声明不赋值，function既声明又赋值。

2.let和const不存在变量提升，并且只有在代码块作用域内有效，不属于顶层作用域属性。

3.使用let命令声明变量之前，该变量都是不可用的。成为暂时性死区

4.let不允许在相同作用域内，重复声明同一个变量

5.const`一旦声明变量，就必须立即初始化，不能留到以后赋值。

1.let

作用：用来声明变量，只在let命令所在的代码块内有效

使用方法：

{
  let a = 10;
  var b = 1;
}
 
a // ReferenceError: a is not defined.
b // 1

特点：

变量提升：

var 存在变量提升；

let不存在变量提升

暂时性死区： 使用let命令声明变量之前，该变量都是不可用的。

！重复声明：let不允许在相同作用域内，重复声明同一个变量

全局变量：

var命令声明属于顶层对象属性

​ let命令声明不属于顶层对象属性

2.const

作用：声明一个只读的常量。一旦声明，常量的值就不能改变。

特点：

初始化：const一旦声明变量，就必须立即初始化，不能留到以后赋值。

作用域：只在声明所在的块级作用域内有效。

全局变量：const命令声明不属于顶层对象属性

变量提升：不存在变量提升

！重复声明：不可重复声明。

将对象和数组声明为常量：可变；

冻结对象：Object.freeze
