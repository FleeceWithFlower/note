## 权限

Linux/Unix 的文件调用权限分为三级 : 文件拥有者、群组、其他。利用 chmod 可以藉以控制文件如何被他人所调用。

常用的 [linux](http://www.111cn.net/list-156/) 文件权限：

444 r--r--r--
600 rw-------
644 rw-r--r--
666 rw-rw-rw-
700 rwx------
744 rwxr--r--
755 rwxr-xr-x
777 rwxrwxrwx

从左至右，1-3位数字代表文件所有者的权限，4-6位数字代表同组用户的权限，7-9数字代表其他用户的权限。

读取的权限等于4，用r表示

写入的权限等于2，用w表示

执行的权限等于1，用x表示

通过4、2、1的组合，得到以下几种权限：0（没有权限）；4（读取权限）；5（4+1 | 读取+执行）；6（4+2 | 读取+写入）；7（4+2+1 | 读取+写入+执行）

### ls

- list 的缩写

> 列出指定目录下的内容及其相关属性信息

参数

- -a

  > 显示所有文件及目录 (包括以“.”开头的隐藏文件)

- -R

  > 递归列出所有子目录 

- -l

  > 使用长格式列出文件及目录信息

- -r

  > 将文件以相反次序显示(默认依英文字母次序)

- -S

  > 根据文件大小排序

### mkdir

- make directories

> 创建目录

### cp

- copy

> 复制文件或目录


参数

- -f

> 若目标文件已存在，则会直接覆盖原文件

- -i

> 若目标文件已存在，则会询问是否覆盖



### mv

- move

> 移动文件或对其改名

- -i

> 若存在同名文件，则向用户询问是否覆盖	

- -f

> 覆盖已有文件时，不进行任何提示

- -b

> 当文件存在时，覆盖前为其创建一个备份
