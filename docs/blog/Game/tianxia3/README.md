### 资料

  蓝色属性：敏，力，魂，念，疾，体。
  橙色属性：水/毒/火/风/元系附伤，水/毒/火/风/元系重击，水/毒/火/风/元系化解，挥砍/穿刺/钝击重击，挥砍/穿刺/钝击化解。
  紫色属性：最大物理攻击，最大法术攻击，最小物理攻击，最小法术攻击，物理防御，法术防御。
  金色属性：追电，骤雨，疾语，扰心，冥思，神明。



::: tip

固定蓝色属性和固定紫色属性的成本相同，固定金色属性和固定橙色属性的成本相同。  

:::
