总结：

1.从数组和对象中提取值，对变量进行赋值，这被称为解构

2.数组的解构赋值，（变量与数组位一一对应），let [a, b, c] = [1, 2, 3];

3.对象的解构赋值，（变量与对象同名属性一一对应），let { bar, foo } = { bar: "bbb"， foo: "aaa"};

4.Rest element 省略赋值 ,let[a,b,...c]=[1,2,3,4],打印出来 a=1,b=2,c为一个数组[3,4]

 

1.数组的解构赋值

方法：（变量与数组位一一对应）

​    

let [a, b, c] = [1, 2, 3];

 

2.对象的解构赋值

方法：（变量与对象同名属性一一对应）

 

let { bar, foo } = { bar: "bbb"， foo: "aaa"};

foo // "aaa"

bar // "bbb"}

 

foo位`foo`是匹配的模式，`baz`是变量，真正被赋值的是变量`baz``。`

 

{let { foo: baz } = { foo: "aaa", bar: "bbb" };

baz // "aaa"

foo // error: foo is not defined}

 

默认值：  

  默认值生效的条件是，对象的属性值严格等于`undefined`。

3.字符串的解构赋值

字符串也可以解构赋值。这是因为此时，字符串被转换成了一个类似数组的对象

```
 
{const [a, b, c, d, e] = 'hello';
a // "h"
b // "e"}
 
```

4.数值和布尔值解构赋值

 规则： 解构赋值时，如果等号右边是数值和布尔值，则会先转为对象。

```
{let {toString: s} = 123;

s === Number.prototype.toString // true

let {toString: s} = true;

s === Boolean.prototype.toString // true
}
特点：由于undefined和null无法转为对象，所以对它们进行解构赋值，都会报错。
```

 

5.函数的参数的解构赋值

```
 
{function add([x, y]){
  return x + y;
}
add([1, 2]); // 3}
```
