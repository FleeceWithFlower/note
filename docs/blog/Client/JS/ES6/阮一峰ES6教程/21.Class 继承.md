1.Class

定义类

构造函数的另一种写法

```
class Point {
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }//构造函数
 
  toString() {
    return '(' + this.x + ', ' + this.y + ')';
  }//添加原型方法
}
```

Object.assign

可以很方便地一次向类添加多个方法。

```
class Point {
  constructor(){
    // ...
  }
}//构造函数
 
 
Object.assign(Point.prototype, {
  toString(){},
  toValue(){}
});//添加多个原型方法
```

特点：

类的内部所有定义的方法，都是不可枚举的

## 静态方法

如果在一个方法前，加上`static`关键字，就表示该方法不会被实例继承，而是直接通过类来调用，这就称为“静态方法”。
