## 标题

- 使用 = 和 - 标记一级和二级标题

```
我展示的是一级标题
=================

我展示的是二级标题
-----------------
```

- 使用 **#** 号可表示 1-6 级标题，

```
# 一级标题
## 二级标题
```

## 表格

- **|** 来分隔不同的单元格
- 使用 **-** 来分隔表头和其他行。

- **-:** 设置内容和标题栏居右对齐。
- **:-** 设置内容和标题栏居左对齐。
- **:-:** 设置内容和标题栏居中对齐。

```
| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```
