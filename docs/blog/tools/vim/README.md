## 三种模式

- **命令模式（Command mode）**
- **输入模式（Insert mode）**
- **底线命令模式（Last line mode）**

1. ### 使用 vi/vim 进入命令模式

2. ### 按下 i 进入输入模式

3. ### 按下 ESC 按钮回到命令模式

4. ### 在命令模式中按下 **:wq** 储存后离开 vi

## 搜索

- ## /word

  > 向下搜索

  n

  > 向下搜索下一個

  N

  > 向上收索上一個

- ## ?word

  > 向上搜索

  n

  > 向下搜索下一個

  N

  > 向上收索上一個
